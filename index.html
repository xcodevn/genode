Welcome to Tinycore/Genode

This page show the way to create a version of linux run on genode framework.

I assumes you know how to install git in your machine.

To clone this repo into your computer:

$ git clone https://github.com/xcodevn/genode.git

Prepare tool-chain

The first thing you need for compiling is tool-chains. For convenient, genode framework sticks with only one tool-chain. You have to choices here: (1) your can download pre-compiled tool-chain from Internet or (2) compiler tool-chain on your machine. For detail information, go there: http://genode.org/download/tool-chain.

If you're lazy for reading these notes and like looking compile progress. Do follow commands:

$ cd genode
$ ./tool/tool_chain x86

Then, have a cup of coffee and waiting ...

Create your image

I intend to create a script for doing all those bellow stuff in automatic way. For current stage, you have to do by your hands:

# Prepare foc
$ cd ~/genode/base-foc
$ make prepare
# Prepare libports
$ cd ~/genode/libports
$ make prepare
# Prepare port
$ cd ~/genode/ports
$ make prepare
# Prapre dde_ipxe
$ cd ~/genode/dde_ipxe
$ make prepare
# Finally, prepare L4linux
$ cd ~/genode/ports-foc
$ make prepare

I hope you have a fast Internet line ;)

After those step, now, we'll create Makefile

$ cd ~/genode
$ ./tool/create_builddir foc_x86_32 tinycore   # new directory 'tinycore' will be created
# Add one more thing
$ cat >> tinycore/etc/build.conf <<EOF
REPOSITORIES += $(GENODE_DIR)/libports
REPOSITORIES += $(GENODE_DIR)/ports
REPOSITORIES += $(GENODE_DIR)/ports-foc
REPOSITORIES += $(GENODE_DIR)/dde_ipxe
# This bellow line for who has a two-cores CPU
MAKE += -j 2
EOF

Finally, run our tinycore

$ cd ~/genode/tinycore
$ make run/tinycore

We have to wait for everything is compiled. Enjoy your hacking!
